# agentic_ai

A minimal React app used as a POC for testing agentic AI automation. It contains a single Login component and nothing else.

## Structure

```
src/
  components/
    Login/
      Login.jsx
  __test__/
    Login.test.jsx
  main.jsx
```

## Scripts

- `npm install` - install dependencies
- `npm run dev` - start the dev server
- `npm run build` - build for production
- `npm test` - run tests
